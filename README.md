# 🏫 School Management System

A simple command-line based School Management System developed in Python to manage student records efficiently.

The system allows users to add, view, and delete student records while storing data permanently using CSV files.

---

## ✨ Features

- Add new student records
- Store student details in CSV format
- Display all student records
- Calculate and display average marks
- Delete student records using Roll Number
- Persistent data storage using CSV files

---

## 📂 Student Information Stored

The system stores the following information:

- Student Name
- Class
- Roll Number
- English Marks
- Mathematics Marks
- Physics Marks
- Chemistry Marks
- Computer Marks
